# Game Kuis tentang Arduino - MotoGP Racer List menggunakan Gemini Canvas

## Deskripsi Proyek

Proyek ini merupakan aplikasi interaktif yang menggabungkan kuis tentang Arduino dengan fitur penampilan daftar nama pembalap MotoGP. Aplikasi ini menggunakan teknologi Gemini Canvas untuk memberikan pengalaman pengguna yang menarik dan responsif.

## Fitur Utama

- 🎮 **Kuis Interaktif** - Kuis tentang Arduino untuk menguji pengetahuan
- 🏍️ **Daftar Pembalap MotoGP** - Menampilkan informasi lengkap tentang pembalap MotoGP
- 🎨 **Gemini Canvas** - Menggunakan Gemini Canvas untuk visualisasi yang menarik
- 💻 **Berbasis Web** - Dibangun menggunakan teknologi HTML

## Teknologi yang Digunakan

- **HTML** - Untuk struktur dan tampilan aplikasi web
- **Gemini Canvas** - Untuk rendering dan visualisasi interaktif
- **Arduino Knowledge Base** - Data dan informasi tentang Arduino

## Cara Menggunakan

1. Clone repository ini
   ```bash
   git clone https://github.com/AlfianZw/Game-Kuis-tentang-Arduino-menampilkan-daftar-nama-pembalap-MotoGP-menggunakan-Gemini-Canvas.git
   ```

2. Buka file HTML di browser Anda
   ```bash
   open index.html
   ```

3. Mulai bermain kuis dan jelajahi daftar pembalap MotoGP
